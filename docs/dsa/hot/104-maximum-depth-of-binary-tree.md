## 思路

使用**递归**计算二叉树的最大深度：

1. 递归终止条件：
   - 如果节点为空，返回深度 0
2. 递归计算：
   - 递归计算左子树的最大深度
   - 递归计算右子树的最大深度
   - 返回左右子树深度的最大值 + 1（当前节点）

### 关键点

- 二叉树的深度 = 左右子树深度的最大值 + 1
- 使用 `Math.max` 比较左右子树深度
- 简洁的递归实现，一行代码完成

- 时间复杂度 $O(n)$，其中 $n$ 是节点数量
- 空间复杂度 $O(h)$，其中 $h$ 是树的高度，递归调用栈的深度
