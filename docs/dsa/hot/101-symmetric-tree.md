## 思路

判断一棵二叉树是否对称，需要比较左右子树是否镜像对称：

1. 递归终止条件：
   - 如果左右节点都为空，返回 true
   - 如果左右节点只有一个为空，返回 false
2. 递归比较：
   - 比较左右节点的值是否相等
   - 递归比较左子树的左孩子和右子树的右孩子
   - 递归比较左子树的右孩子和右子树的左孩子

### 关键点

- 对称比较：左子树的左孩子 vs 右子树的右孩子，左子树的右孩子 vs 右子树的左孩子
- 使用辅助函数 `symmetric` 进行递归比较
- 需要同时满足值相等和子树对称

- 时间复杂度 $O(n)$，其中 $n$ 是节点数量
- 空间复杂度 $O(h)$，其中 $h$ 是树的高度，递归调用栈的深度
