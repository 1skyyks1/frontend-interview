## 思路

使用**模拟 + 方向数组**的方法按螺旋顺序遍历矩阵：

1. **方向数组**：定义四个方向 `→ ↓ ← ↑`，分别对应右、下、左、上
2. **访问标记**：使用 `visited` 二维数组标记已访问的位置
3. **螺旋遍历**：
   - 从 `(0, 0)` 开始，按当前方向前进
   - 每走一步，将当前位置的值存入结果数组，并标记为已访问
   - 尝试下一步时，如果越界或已访问，则顺时针转向（方向索引 `+1 % 4`）
   - 继续前进直到遍历完所有元素

这种方法直观地模拟了螺旋遍历的过程。

- 时间复杂度 $O(m \times n)$，其中 $m$ 为矩阵行数，$n$ 为矩阵列数
- 空间复杂度 $O(m \times n)$，需要额外的 `visited` 数组和结果数组
